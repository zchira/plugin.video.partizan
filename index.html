About

Kodi/XBMC plugin for watching FK Partizan official TV station. FK Partizan is part of JSD Partizan (Yugoslav Sports Association Partizan).

How to install

This plugin (and some other) may be installed from LazyRobot Kodi plugin repository. Download repository archive and instal it from Kodi (Video->Add on->Install from Zip).
